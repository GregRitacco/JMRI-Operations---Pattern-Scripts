Description of PatternConfig.JSON file
  "ControlPanel" : { These are settings used for the main script
    "TrackPattern" : true, True means use the track pattern scripts - not implemented
    "PH" : 140, Panel Height
    "PW" : 2000 Pane Width
  },
  "TP" : { Track Pattern script values
    "PL" : "", Pattern Location, a valid JMRI location
    "PT" : { Pattern Tracks, a list of tracks for the valid location and their selected status
    },
    "PA" : false, Yard Tracks Only flag
    "PI" : false, Ignore track length flag
    "PW" : 260, Pattern Width, text box width
    "PH" : 16, Pattern Height, height of all pattern elements
    "ST" : "Set to:", Input box form header
    "SL": [ Sort List, the switch lists are sorted on these items
      "Load Type",
      "FD&Track"
    ],
    "RM": 8, Report Multiplier
