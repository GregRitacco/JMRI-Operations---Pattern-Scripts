Page d'aide - plugin Pattern Scripts pour JMRI

Introduction

Ce programme est un plugin pour JMRI Operations Pro. Avec lui, l'utilisateur peut générer des rapports de motifs pour les voies d'un même emplacement. En outre, les wagons peuvent être déplacés d'une voie à l'autre au sein d'un emplacement et, en option, si un wagon est déplacé sur un embranchement, l'horaire de cet embranchement peut être appliqué. Le support de ma suite de scripts TrainPlayer est également inclus. L'utilisation de ce plugin est documentée dans la liste de vidéos suivante.

Modèle de piste et liste d'aiguillage en format JSON

Les rapports de modèle et les exportations TrainPlayer commencent tous par la création d'un fichier JSON. Le fichier est créé par ce plugin et dans le cas des exportations du manifeste TrainPlayer, le JSON est créé en modifiant le JSON du manifeste généré par JMRI. Tous les fichiers JSON sont écrits dans le dossier jsonManifests. Ils suivent le format illustré:

Clé – PatternConfig.json

Chaque profil JMRI obtient son propre fichier de configuration, qui s'appelle PatternConfig.json. Il se trouve dans le dossier des opérations, et est utilisé pour fournir des paramètres personnalisés pour chaque profil et enregistrer les entrées des utilisateurs.
La description de chaque élément est en bleu.
Les touches modifiables par l'utilisateur sont décrites en vert.
Des garde-fous en quelque sorte ont été mis en place autour des valeurs marquées en vert, mais malgré cela, ce plugin n'est pas en mesure d'attraper tous les types d'erreurs. Dans certains cas, le plugin peut planter. Si cela se produit, appuyez sur le bouton " Restart from PatternConfig Backup " et le plugin utilisera le dernier bon fichier PatternConfig.json comme nouveau point de départ. Dans d'autres cas, si une erreur est commise en modifiant le fichier PatternConfig.json, le plugin le remplacera par le fichier par défaut, et l'utilisateur devra recommencer.
Dans le pire des cas, il suffit de supprimer le fichier Pattern Config.json et le plugin en créera un nouveau.
